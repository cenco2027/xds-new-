## 🎯 响应式断点策略

### 断点定义

```css
/* Mobile First Approach */
/* Mobile: 0-767px (默认) */
/* Tablet: 768px-1023px */
/* Tablet Exact: 1024px */
/* Desktop: 1025px-1727px */
/* Desktop Large: 1728px+ */
```

### 断点使用建议

1. **Mobile (< 768px)**
   - 使用最小尺寸和间距
   - 纵向布局为主
   - 隐藏非必要元素

2. **Tablet (768px - 1024px)**
   - 中等尺寸和间距
   - 纵向布局为主
   - 1024px时使用精确的Tablet设计

3. **Desktop (1025px - 1727px)**
   - 使用流体布局（clamp）
   - 横向布局为主
   - 平滑过渡

4. **Desktop Large (≥ 1728px)**
   - 使用固定尺寸
   - 最大宽度限制
   - 精确还原设计

## 🔄 状态变化说明

### 筛选标签状态

**默认状态**:
- 边框: 2px solid Text 1
- 背景: transparent
- 文字: Text 1 (#F4F6F6)

**激活状态**:
- 边框: 2px solid Text 1
- 背景: Text 1 (#F4F6F6)
- 文字: Bg Black 1 (#23262B)

**悬停状态**:
- 添加透明度变化
- 过渡动画: 0.3s ease

### 产品卡片显示逻辑

1. **默认显示**: 显示所有产品（或根据初始筛选）
2. **筛选后**: 只显示匹配筛选标签的产品
3. **过渡动画**: 卡片淡入淡出效果
