## 🔧 下载方法

### 使用Figma MCP工具下载

#### 步骤1: 准备下载列表
创建包含所有需要下载的图片的节点列表

#### 步骤2: 下载背景图片
```javascript
// 示例: 下载桌面端Hero背景
{
  fileKey: "6xRC7csnzLdkfC8wTuWOxv",
  nodes: [
    {
      nodeId: "3390:1284",
      fileName: "hero-bg-desktop.png",
      imageRef: "110a69ce2f4d30b1b3d1e1e9c0b5497120a636df",
      needsCropping: false
    }
  ],
  localPath: "D:/work/xds new/assets/images/contact",
  pngScale: 2
}
```

#### 步骤3: 下载图标 (SVG优先)
```javascript
// 示例: 下载下拉箭头图标
{
  fileKey: "6xRC7csnzLdkfC8wTuWOxv",
  nodes: [
    {
      nodeId: "3390:1312",
      fileName: "dropdown-arrow.svg",
      // SVG不需要imageRef
```

## 📝 注意事项

1. **图片格式选择**:
   - Logo和图标: 优先使用SVG格式
   - 背景图片: 使用PNG格式
   - 可以考虑使用WebP格式优化性能

2. **图片尺寸**:
   - 桌面端: 2x分辨率 (使用pngScale: 2)
   - 平板端: 2x分辨率
   - 移动端: 根据实际需要

3. **图片优化**:
   - 下载后可能需要压缩
   - 考虑使用图片压缩工具
   - 移动端背景图片需要裁剪以减小文件大小

4. **Logo复用**:
   - 确认现有的Logo文件是否可以复用
   - 如果已存在，只需确认尺寸是否符合要求
